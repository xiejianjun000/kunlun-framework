## 一、产品定位与设计理念

### 1.1 产品定位
「太极·观星台」是 OpenTaiji 多智能体框架的统一可视化管理控制中心，为运维人员、开发者、企业管理员提供一站式的智能体集群管理、监控、调优能力。

### 1.2 设计理念：东方哲思 × 现代交互

**核心Slogan**: 「运筹帷幄之中，决胜千里之外」

**文化辨识度锚点**:
- **八卦布局**: 仪表盘采用八卦方位布局8大核心功能入口
- **太极健康度**: 系统状态采用阴阳鱼可视化，绿色=阳=健康，红色=阴=异常
- **五行配色**: 金(白)·木(绿)·水(蓝)·火(红)·土(黄) 对应不同功能域
- **东方术语体系**:
  - 版本升级 → 「修炼进阶」
  - 系统初始化 → 「开天辟地」
  - 智能体集群 → 「众生相」
  - 技能市场 → 「藏经阁」
  - 日志中心 → 「因果录」
  - 调度任务 → 「天机漏」
  - WFGY监控 → 「照妖镜」
  - 系统设置 → 「乾坤鼎」

### 🎯 Module 1: 太极总览仪表盘 - 「观星台」

**页面定位**: 系统运行状态一屏掌控，健康度一目了然

#### 1.1 核心指标卡片（4个，2×2布局）

| 卡片名称 | 显示内容 | 交互 |
|---------|---------|------|
| 🧠 智能体在线数 | 「X / Y 在线」<br>当前运行中 / 总智能体数<br>趋势箭头（↑↓） | 点击跳转「智能体管理」 |
| ⚡ 技能执行次数 | 今日 / 累计<br>QPS实时数值<br>趋势折线小图 | 点击跳转「调度任务」 |
| 💰 Token消耗统计 | 今日消耗 / 累计消耗<br>预估费用（可选） | 点击跳转「模型适配器中心」 |
| 🛡️ WFGY防护统计 | 幻觉检出率: X%<br>已拦截次数: X次 | 点击跳转「WFGY防幻觉监控」 |

#### 1.2 太极健康度指示器（页面顶部中央）

**视觉**: 动态旋转的太极阴阳鱼
- 🟢 全绿太极 = 系统100%健康
- 🟡 半黄半绿 = 存在警告，不影响核心功能
- 🔴 半红半绿 = 存在严重错误，需要立即处理
- 🌀 旋转速度 = 系统负载程度（快=高负载，慢=空闲）

**交互**: 点击太极图展开健康度详情面板，显示各子系统健康分

#### 1.3 核心图表区域

| 图表 | 类型 | 数据维度 |
|------|------|---------|
| 技能执行QPS趋势 | 折线图 | 最近24小时，按5分钟粒度 |
| 任务执行成功率 | 饼图 | 成功 / 失败 / 重试中 |
| 模型调用分布 | 柱状图 | 7大国产模型各自调用占比 |

#### 1.4 实时日志流（页面底部）

- 最近10条系统日志，自动滚动
- 按日志级别着色：DEBUG灰/INFO蓝/WARNING黄/ERROR红
- 可暂停滚动
- 点击「查看全部」跳转日志中心

#### 1.5 快捷操作栏（页面右上角）

- ➕ 新建智能体
- ⚙️ 全局设置
- 🔔 告警中心（红点显示未读告警数）
- 👤 用户头像与菜单

## 四、完整设计规范

### 4.1 配色系统（五行配色）

| 功能域 | 主色 | 对应五行 |
|-------|------|---------|
| 健康/成功 | `#10b981` 翡翠绿 | 木 |
| 警告/注意 | `#f59e0b` 琥珀黄 | 土 |
| 错误/危险 | `#ef4444` 朱红 | 火 |
| 信息/链接 | `#3b82f6` 天蓝 | 水 |
| 中性/背景 | `#f8fafc` 云白 | 金 |

### 4.2 组件规范

- 统一使用圆角：`12px` 卡片圆角，`8px` 按钮圆角
- 阴影层级：浮层 `shadow-xl`，卡片 `shadow-md`，按钮 `shadow-sm`
- 字体层级：标题 `20px 600`，副标题 `16px 500`，正文 `14px 400`，辅助 `12px 400`
- 所有hover状态有明确反馈

### 4.3 动画规范

- 页面切换：淡入淡出 `300ms ease`
- 抽屉滑入：从右往左 `300ms cubic-bezier(0.4, 0, 0.2, 1)`
- 太极旋转：匀速 `20s/r`，健康度变化过渡 `500ms ease`
