## 🚀 快速开始文案

### 三步上手 OpenTaiji

#### 第一步：安装

```bash
npm install @opentaiji/core
```

> 一行命令，从零到一。

#### 第二步：创建你的第一个 Actor

```typescript
import { OpenTaiji, Actor, Message } from "@opentaiji/core";

const taiji = await OpenTaiji.start();

class HelloActor extends Actor {
  async receive(message: Message) {
    console.log(`👋 收到：${message.payload}`);
  }
}

const actor = await taiji.spawnActor(HelloActor);
actor.send({ payload: "Hello, OpenTaiji!" });
```

> 五个类，十行代码，你的第一个智能体就活了。

#### 第三步：注册技能，开始施法

```typescript
import { Skill, SkillContext } from "@opentaiji/core";

class WeatherSkill extends Skill {
  name = "weather";
  
  async execute(city: string, context: SkillContext) {
    return `今天${city}的天气是晴天，25°C ☀️`;
  }
}

taiji.registerSkill(new WeatherSkill());
const result = await taiji.executeSkill("weather", "北京");
```

> 定义你的能力，剩下的交给 OpenTaiji。

## 📝 常见问题 FAQ 文案

### Q: OpenTaiji 和 LangChain / LlamaIndex 有什么区别？

**A:** LangChain 是工具箱，OpenTaiji 是操作系统。

LangChain 给你一堆积木，你要自己搭房子、接水电、做装修。OpenTaiji 给你一个已经建好的、精装修的、可以直接入住的房子，你只需要搬进去住就行。

简单说：用 LangChain 你要自己解决并发、容错、监控、多租户、记忆管理等等所有问题。用 OpenTaiji，这些都已经帮你解决好了。

### Q: 单机可以支撑多少并发？

**A:** 取决于你的硬件，但可以给你一个参考。

在一台 16 核 32G 的服务器上：
- 同时在线 10,000+ 活跃 Actor
- 每秒处理 5,000+ 条消息
- 内存占用 < 16GB
- P99 延迟 < 50ms

集群模式下可以水平扩展到百万级并发。
