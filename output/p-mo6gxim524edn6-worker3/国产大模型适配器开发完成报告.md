## 设计特点

### 1. 统一接口 `ILLMAdapter`

所有适配器都实现相同接口，支持热切换：

```typescript
interface ILLMAdapter {
  readonly name: string;
  readonly version: string;
  readonly model: string;
  
  isReady(): boolean;
  complete(messages: LLMMessage[], options?): Promise<LLMCompletionResult>;
  completeStream(messages: LLMMessage[], options?): AsyncGenerator<string>;
  getTokenUsage(): TokenUsage;
  resetTokenUsage(): void;
  calculateCost(promptTokens: number, completionTokens: number): number;
}
```

### 2. 基类`BaseLLMAdapter`提供通用功能

- ✅ 统一token累计统计
- ✅ 统一成本计算
- ✅ 带指数退避的重试机制 (`fetchWithRetry`)
- ✅ 统一错误处理
- ✅ 减少重复代码

### 3. 失败回退机制

- `maxRetries` 可配置最大重试次数，默认3次
- 指数退避延迟：`2^retries * 1000 + random` 避免风暴
- 超时控制：默认60秒超时，可配置

### 4. 成本统计功能

- 累计记录 `promptTokens` / `completionTokens` / `totalTokens`
- 根据每个模型价格计算估算成本
- 支持随时重置统计

## 使用示例

```typescript
import { QwenAdapter } from './adapters/llm';

const adapter = new QwenAdapter({
  apiKey: process.env.DASHSCOPE_API_KEY!,
  model: 'qwen-turbo',
  costPer1kPrompt: 0.0008,
  costPer1kCompletion: 0.0008
});

const messages = [
  { role: 'user', content: 'Hello, world!' }
];

const result = await adapter.complete(messages, {
  temperature: 0.7,
  maxTokens: 1000
});

console.log(result.text);
console.log('Cost:', adapter.getTokenUsage().estimatedCostUsd);
```

## 热切换示例

```typescript
import type { ILLMAdapter } from './adapters/llm/interfaces/ILLMAdapter';
import { QwenAdapter } from './QwenAdapter';
import { DoubaoAdapter } from './DoubaoAdapter';

// 可以根据配置动态选择
let adapter: ILLMAdapter;
if (provider === 'qwen') {
  adapter = new QwenAdapter(config);
} else if (provider === 'doubao') {
  adapter = new DoubaoAdapter(config);
}

// 使用统一接口调用，不需要修改调用代码
const result = await adapter.complete(messages);
```
