## 📝 适配器特性说明

### SenseNova (商汤日日新)
- **API 端点**: `https://api.sensenova.cn/v1/chat/completions`
- **支持模型**: sensenova-pro, sensenova-lite
- **计费策略**: ￥0.01 / 1K tokens (pro)
- **鉴权方式**: Bearer Token

### Pangu (华为云盘古)
- **API 端点**: 华为云区域化端点
- **支持模型**: pangu-7b, pangu-70b, pangu-finance
- **计费策略**: ￥0.008~0.04 / 1K tokens
- **特殊配置**: 需要 Project ID
- **鉴权方式**: HMAC-SHA256 签名

### Yuyan (网易玉言)
- **API 端点**: `https://llm.youdao.com/v1/chat`
- **支持模型**: yuyan-v2-34b, yuyan-v2-7b
- **计费策略**: ￥0.005~0.01 / 1K tokens
- **鉴权方式**: API Key + 时间戳签名

### Xiaoai (小米小爱)
- **API 端点**: `https://api.xiaoai.mi.com/v1/chat`
- **支持模型**: xiaoai-pro, xiaoai-lite
- **计费策略**: ￥0.003~0.008 / 1K tokens
- **鉴权方式**: Bearer Token

### Hongshan (出门问问红杉)
- **API 端点**: `https://api.aimind.com/v1/chat`
- **支持模型**: hongshan-pro, hongshan-lite
- **计费策略**: ￥0.005~0.01 / 1K tokens
- **鉴权方式**: API Key Header

### Xuanyuan (度小满轩辕)
- **API 端点**: `https://api.xuanyuan.duxiaoman.com/v1/chat`
- **支持模型**: xuanyuan-finance, xuanyuan-general
- **计费策略**: ￥0.015~0.025 / 1K tokens
- **鉴权方式**: Bearer Token + App ID

### Tiangong (昆仑万维天工)
- **API 端点**: `https://api.tiangong.cn/v1/chat/completions`
- **支持模型**: tiangong-pro, tiangong-lite
- **计费策略**: ￥0.005~0.01 / 1K tokens
- **鉴权方式**: Bearer Token

## 🎯 使用示例

```typescript
import { LLMAdapterFactory } from './src/adapters/llm';

// 创建商汤日日新适配器
const adapter = LLMAdapterFactory.create('sensenova', {
  apiKey: process.env.SENSENOVA_API_KEY,
  model: 'sensenova-pro',
});

// 初始化
await adapter.initialize();

// 调用对话
const response = await adapter.createChatCompletion({
  messages: [
    { role: 'user', content: '你好，请介绍一下自己' }
  ],
  temperature: 0.7,
});

console.log(response.content);
console.log('本次调用成本:', adapter.getTotalUsage().cost);
```
