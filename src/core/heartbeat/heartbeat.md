# 心跳检查清单模板

> 此文件由 HeartbeatManager 自动读取，用于周期性巡检
>
> 触发频率：每30分钟一次

---

## 检查清单

- [ ] 检查最近对话是否符合SOUL人设
- [ ] 检查是否有工具调用失败超过3次
- [ ] 检查记忆文件是否有矛盾内容
- [ ] 检查是否有未完成的任务
- [ ] 检查系统资源使用情况

---

## 示例检查项

### 1. 人设合规检查
```
检查：最近3条回复是否符合"咔咔"的人设风格？
- 说话是否简洁？
- 是否避免了机械式回复？
- 是否有个性化的表达？
```

### 2. 工具调用检查
```
检查：是否有连续失败的工具调用？
- 统计最近失败的调用次数
- 是否超过阈值（默认3次）
- 是否需要通知用户
```

### 3. 记忆污染检查
```
检查：记忆文件是否有矛盾或污染内容？
- 是否有重复存储的记忆
- 是否有冲突的信息
- 是否需要清理
```

### 4. 任务完成检查
```
检查：是否有长时间未完成的任务？
- 检查calendar中的日程任务
- 检查sub-agent状态
- 是否需要提醒用户
```

### 5. 系统健康检查
```
检查：系统资源使用情况
- 内存使用率
- 磁盘空间
- 数据库连接
```
