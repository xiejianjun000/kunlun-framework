# 昆仑框架 - 开源中国推广文章

## 项目介绍

**昆仑框架（Kunlun Framework）** 是基于 Actor 模型 + LLM 对话引擎的多智能体开源框架，专为环保、工业、医疗等领域设计。

## 核心功能

- 🏗️ **Actor模型**：原生支持并发，轻松应对千人级访问
- 🤖 **十三神智能体**：环保领域开箱即用的13个专业化智能体
- 🧬 **技能进化**：智能体可自主学习进化
- 💓 **心跳自检**：三道防线保障系统稳定
- 🏢 **多租户架构**：企业级隔离与计费
- 🎭 **人格蒸馏**：五维画像系统

## 技术栈

- TypeScript 5.0+
- Actor Runtime
- PostgreSQL + Qdrant + Redis + Kafka + Neo4j

## 快速开始

```bash
npm install kunlun-framework
```
