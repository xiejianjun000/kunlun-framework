## 二、昆仑框架核心特性

### 1. Actor模型：并发处理的工业级方案

昆仑框架采用经典的Actor模型作为并发基础，每个智能体都是独立的"演员"：

```python
from kunlun import Actor, Message

class EnvironmentalMonitor(Actor):
    """环境监测智能体"""
    
    async def receive(self, message: Message) -> Message:
        data = message.content
        # 处理监测数据
        result = await self.analyze_air_quality(data)
        return Message(content=result, recipient="coordinator")
    
    async def analyze_air_quality(self, data: dict):
        # 模拟AQI分析逻辑
        aqi = self.calculate_aqi(data)
        return {"station": data["id"], "aqi": aqi, "level": self.get_level(aqi)}
```

**Actor模型的核心优势：**
- ✅ **天然并发**：每个Actor独立处理消息，无需锁机制
- ✅ **状态隔离**：Actor之间不共享状态，避免竞态条件
- ✅ **位置透明**：可轻松扩展到分布式部署
- ✅ **容错设计**：支持监督树（Supervision Tree）自动恢复

### 3. 技能进化：让智能体越用越聪明

昆仑框架内置**自我进化机制**，智能体可以从任务执行中持续学习：

```python
from kunlun import EvolutionEngine

# 启用技能进化
evolution = EvolutionEngine(
    learning_rate=0.01,
    min_samples=100,
    improvement_threshold=0.15
)

# 智能体执行任务后自动记录
evolution.record_experience(
    agent_id="risk_warning",
    task_result=result,
    feedback=feedback,
    context=task_context
)

# 触发进化评估
if evolution.should_evolve("risk_warning"):
    new_capability = evolution.evolve("risk_warning")
    await agent.update_capability(new_capability)
```

**进化机制亮点：**
- 📈 **增量学习**：从成功案例中提取最优策略
- 🎯 **自适应阈值**：自动调整告警敏感度
- 🔄 **版本回滚**：进化失败可一键恢复
- 📊 **可视化仪表盘**：追踪每个智能体的进化轨迹

### 4. 多租户架构：企业级隔离与计费

```python
from kunlun import TenantContext, IsolationPolicy

# 多租户配置
tenant_config = {
    "tenant_id": "eco_company_a",
    "quota": {
        "max_agents": 50,
        "max_messages_per_day": 100000,
        "storage_gb": 100

    "isolation": IsolationPolicy.STRICT,

        "model": "tiered",
        "base_rate": 0.001,  # per message

    }

# 租户隔离执行
async with TenantContext(tenant):
    result = await agent.execute(task)
```

## 三、实战：5分钟构建环境监测系统

### 安装与初始化

```bash
pip install kunlun-framework

# 初始化项目
kunlun init environmental-system
cd environmental-system
```

### 定义监测流程

```python
# app.py
from kunlun import KunlunApp, Pipeline
from kunlun.agents import DataCollector, Analyzer, WarningAgent, Reporter

app = KunlunApp(
    name="Environmental Monitoring",
    agents=[
        DataCollector(),
        Analyzer(),
        WarningAgent(threshold=150),  # AQI阈值
        Reporter(template="weekly_report")
    ]
)

# 定义数据流
pipeline = Pipeline()
pipeline.add_stage("collect", DataCollector)
pipeline.add_stage("analyze", Analyzer)
pipeline.add_stage("warn", WarningAgent, condition="anomaly_detected")
pipeline.add_stage("report", Reporter)

app.set_pipeline(pipeline)

# 启动服务
app.serve(port=8080)
```

### 启动与测试

```bash
# 启动服务
kunlun run app:app

# 测试数据接入
curl -X POST http://localhost:8080/api/data \
  -H "Content-Type: application/json" \
  -d '{"station_id": "BJ_001", "pm25": 85, "pm10": 120, "no2": 45}'
```
