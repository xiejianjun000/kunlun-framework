# OpenTaiji Web 界面预览

本目录包含OpenTaiji Web 聊天界面的设计预览图。

## 界面预览

### 1. 智能体选择界面
![智能体选择界面](./interface-preview.jpg)

**主要功能:**
- 左侧边栏展示"十三神"智能体列表
- 每个智能体显示头像、名称、描述和在线状态
- 支持搜索过滤智能体
- 显示当前在线智能体数量

### 2. 聊天对话界面
![聊天对话界面](./chat-preview.jpg)

**主要功能:**
- 与选中智能体进行实时对话
- 用户消息（紫色气泡）与助手消息（灰色气泡）区分显示
- 底部输入框支持多行输入
- 实时显示消息发送状态

## 设计特点

- **深色主题**: 护眼的深蓝色配色方案
- **紫色强调**: 主色调为现代科技紫色
- **状态指示**: 绿色=在线, 黄色=忙碌, 灰色=离线
- **圆角卡片**: 现代UI设计风格
- **流畅动画**: 消息滑入、状态切换等动效

1. 直接在浏览器中打开 `.jpg` 文件查看
