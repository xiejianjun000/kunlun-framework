# 非暴力沟通（NVC）

## 概述

非暴力沟通（Nonviolent Communication）是由马歇尔·卢森堡博士开发的沟通方法，通过同理心和清晰表达减少冲突、增进理解。

## 四个要素

### 1. 观察（Observation）
```
描述实际发生的事情
不带评判、不带解释
像摄像机一样客观

❌ 评价："你总是迟到"
✅ 观察："这周你有3次会议迟到"

要点：
- 基于具体事实
- 不带形容词
- 不带评价
```

### 2. 感受（Feeling）
```
表达自己的情绪
不是想法，不是判断

示例：
- 开心、难过、愤怒、焦虑
- 兴奋、失望、担心、疲惫

❌ 想法："我觉得被忽视了"
✅ 感受："我感到孤单"

要点：
- 用情绪词汇
- 区分感受和想法
```

### 3. 需要（Need）
```
感受背后的需求
人类共同的需求

常见需求：
- 安全感
- 尊重
- 理解
- 自主
- 连接
- 意义

示例：
"我需要被尊重"
"我需要安全感"
"我需要被理解"
```

### 4. 请求（Request）
```
具体、明确、可执行的请求
不是命令

❌ 模糊："你改改吧"
✅ 具体："下次开会请你准时5分钟"

❌ 命令："你必须..."
✅ 请求："你可以..."

要点：
- 具体行为
- 可执行
- 可衡量
- 尊重对方选择
```

## 完整表达

### 公式
```
当我看到[观察]
我感到[感受]
因为我需要[需要]
所以你可否[请求]
```

### 示例
```
"当我看到这周有3次会议迟到（观察），
我感到焦虑（感受），
因为我需要尊重团队的时间（需要），
所以你可否下次准时5分钟（请求）？"
```

## 倾听他人

### 四个要素（对他人）
```
1. 他人观察到了什么？
2. 他人有什么感受？
3. 他人有什么需要？
4. 他人想要什么请求？
```

### 同理倾听
```
放下判断
全神贯注
体会对方感受和需要
确认理解
```

### 确认理解
```
"你是说...吗？"
"你感到...因为你需要...对吗？"
```

## 应用场景

### 工作沟通
```
场景：下属工作未完成

❌ 暴力：
"你怎么又没完成？太不靠谱了！"

✅ 非暴力：
"我看到项目没有按时完成（观察），
我感到担心（感受），
因为我需要信任团队能交付（需要），
你可以告诉我是什么原因吗（请求）？"
```

### 家庭沟通
```
场景：孩子不写作业

❌ 暴力：
"你怎么这么懒！快去写作业！"

✅ 非暴力：
"我看到你一直在看电视（观察），
我感到焦虑（感受），
因为我希望你养成良好的学习习惯（需要），
我们约定一下作业时间好吗（请求）？"
```

### 情侣沟通
```
场景：伴侣晚归

❌ 暴力：
"你从来不在乎我！"

✅ 非暴力：
"看到你连续三天很晚回家（观察），
我感到孤单（感受），
因为我需要和你有更多相处时间（需要），
你可以周末陪我出去吗（请求）？"
```

## 常见错误

### 错误1：观察变成评价
```
❌ "你很懒"
✅ "你今天没做家务"
```

### 错误2：想法当感受
```
❌ "我觉得你很自私"
✅ "我感到失望"
```

### 错误3：请求变命令
```
❌ "你必须..."
✅ "你可否..."
```

### 错误4：忽视对方感受
```
只表达自己
不倾听对方
单向沟通
```

## 实践步骤

### 初学阶段
```
1. 区分观察和评价
2. 识别真实感受
3. 连接背后需要
4. 提出具体请求
```

### 进阶阶段
```
1. 快速识别暴力语言
2. 及时转换非暴力
3. 同理倾听他人
4. 帮助他人表达
```

### 高级阶段
```
1. 内化非暴力沟通
2. 冲突现场应用
3. 帮助他人学习
4. 传递非暴力文化
```

## 冲突解决

### 使用非暴力沟通
```
1. 表达自己的观察和感受
2. 倾听对方的观察和感受
3. 识别彼此的需要
4. 共同寻找解决方案
```

### 示例对话
```
A："你从不听我说话！"
NVC转换："我看到你在我说话时看手机（观察），
我感到不被重视（感受），
因为我需要被倾听和理解（需要），
你可以放下手机听我说完吗（请求）？"

B："你总是指责我！"
NVC转换："我听到你感到受伤（感受），
你希望被理解而不是被指责（需要），
你想先说说你的感受吗（请求）？"
```

## 自我同理

### 对自己使用
```
1. 我做了什么/没做什么？（观察）
2. 我对自己有什么感受？
3. 我有什么需要？
4. 我可以如何满足这个需要？
```

### 自我宽恕
```
接受自己的不完美
理解自己的行为
原谅自己的错误
继续向前
```

## 练习建议

### 日常练习
```
- 每天记录一次观察
- 识别和表达感受
- 连接感受和需要
- 提出清晰请求
```

### 伙伴练习
```
- 找练习伙伴
- 互相反馈
- 角色扮演
- 真实场景应用
```

### 反思改进
```
- 复盘冲突沟通
- 识别暴力语言
- 尝试转换
- 持续改进
```
