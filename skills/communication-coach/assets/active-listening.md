## 反馈技巧

### 确认反馈
```
复述确认：
复述内容：
"我听到您说..."
"您的意思是..."
"让我确认一下..."

复述目的：
- 确认理解
- 给对方确认机会
- 显示重视

示例：
对方："这个项目进度太慢了，客户很不满意。"
你："我听到您说项目进度慢导致客户不满，是这样吗？"

澄清确认：
澄清问题：
"您是指..."
"能具体说说..."
"能举个例子..."

澄清目的：
- 消除模糊
- 明确细节
- 深入理解

示例：
"您说的'进度慢'是指比计划晚了一周，还是更长时间？"
```

### 情感反馈
```
情感命名：
"听起来您感到..."
"我感觉到..."
"您似乎..."

情感目的：
- 确认情感
- 表达理解
- 建立连接

示例：
"听起来您对这件事感到很沮丧。"
"我能感受到您对这个结果很失望。"

情感确认：
"任何人遇到这种情况都会..."
"有这种感觉是正常的..."
"我理解您的..."

示例：
"任何人遇到这种情况都会感到沮丧，这很正常。"
```

### 总结反馈
```
总结要点：
"让我总结一下..."
"我们讨论了..."
"主要几点是..."

总结结构：
1. 要点一
2. 要点二
3. 要点三

示例：
"让我总结一下我们讨论的要点：
1. 项目进度比计划晚了两周
2. 客户表达了强烈不满
3. 需要制定补救计划

我理解对吗？"
```

### 提问反馈
```
开放提问：
"能具体说说..."
"您怎么看..."
"还有什么..."

澄清提问：
"您是指..."
"具体是..."
"能举例吗..."

深入提问：
"为什么..."
"怎么..."
"如果..."

探索提问：
"您最关心的是..."
"您的期望是..."
"您的担心是..."
```
