# 谈判框架模板

## 概述

谈判是解决分歧、达成协议的沟通过程。有效的谈判框架能帮助准备充分、执行有效、达成理想结果。

## BATNA分析

### BATNA定义
```
BATNA (Best Alternative to a Negotiated Agreement):
最佳替代方案

重要性：
- 谈判底气
- 决策参考
- 不愿接受底线

判断标准：
- 如果不达成协议
- 我会选择的最佳方案
```

### BATNA开发
```
步骤1：识别替代方案
- 不与对方合作
- 其他合作伙伴
- 自己做
- 暂时搁置

步骤2：评估替代方案
- 可行性分析
- 成本收益分析
- 风险评估

步骤3：选择最佳方案
- 比较各方案
- 确定BATNA

步骤4：改善BATNA
- 提升替代方案
- 增加选择
- 提高BATNA
```

### BATNA应用
```
谈判前：
- 评估BATNA
- 确定底线
- 增加筹码

谈判中：
- 暂露BATNA（谨慎）
- 展示实力
- 不接受劣于BATNA

谈判破裂：
- 启用BATNA
- 重新评估
- 重新谈判
```

## 谈判执行框架

### 开场阶段
```
建立氛围：
- 寒暄破冰
- 积极态度
- 专业形象

明确议程：
- 讨论主题
- 时间安排
- 规则说明

探索阶段：
- 开放式问题
- 了解对方
- 试探底线

核心话术：
- "让我们先确认一下今天的议程..."
- "我想先了解贵方的需求..."
```

### 探索阶段
```
信息收集：
- 开放式提问
- 倾听观察
- 识别模式

利益探索：
- 需求是什么
- 恐惧是什么
- 机会在哪里

核心话术：
- "能否详细说说..."
- "您最关心的是..."
- "我注意到..."
```

### 讨价还价
```
让步技巧：
- 小幅让步
- 交换让步
- 有条件让步

让步原则：
- 交换让步
- 逐步让步
- 不轻易让步

核心话术：
- "如果...那么..."
- "我可以在...方面让步，但..."
- "作为交换..."
```

### 破局处理
```
识别僵局类型：
- 真僵局 vs 假僵局

破解方法：
- 暂停休息
- 转换话题
- 引入新信息
- 拆分问题
- 调整立场

核心话术：
- "我们似乎在...上卡住了"
- "也许我们可以先讨论..."
- "让我们换个角度..."
```

### 收尾阶段
```
确认协议：
- 总结共识
- 明确条款
- 确认细节

记录协议：
- 书面确认
- 双方签字
- 执行计划

关系维护：
- 表达感谢
- 保持关系
- 长期视角

核心话术：
- "让我总结一下我们的共识..."
- "我们还需要确认..."
- "感谢您的..."
```

## 谈判话术

### 倾听话术
```
积极倾听：
- 全神贯注
- 不打断
- 记录要点

反馈倾听：
- 重述确认
- 理解回应
- 澄情回应

核心话术：
- "我听到您说..."
- "您的意思是..."
- "您最关心的是..."
```

### 提问话术
```
开放提问：
- "能否谈谈..."
- "您怎么看..."
- "什么因素..."

澄清提问：
- "您是指..."
- "能详细说明吗..."

假设提问：
- "如果...会怎样..."
- "假设...您会..."

确认提问：
- "所以您是说..."
- "这是否意味着..."
```

### 说服话术
```
利益说服：
- "这对您的好处..."
- "这个方案能解决..."

证据说服：
- "数据显示..."
- "研究表明..."

案例说服：
- "我们的客户..."
- "类似情况下..."
```

### 反对话术
```
转移焦点：
- "这个问题我们可以稍后讨论"

延迟回应：
- "我需要考虑一下"

质疑前提：
- "这个假设是基于..."

拒绝技巧：
- "这不在我的权限内"
```

## 不同类型谈判

### 价格谈判
```
准备：
- 成本分析
- 价格范围
- 非价格价值

策略：
- 先谈价值，后谈价格
- 整体打包
- 创造选择

话术：
- "如果价格太高，我们也可以调整其他要素..."
- "除了价格，我们也需要考虑..."
```

### 合同谈判
```
重点：
- 条款清晰
- 责权明确
- 风险分配

策略：
- 从易到难
- 整体协议
- 分步讨论

话术：
- "关于这一条，我建议..."
- "我们是否可以..."
```

### 冲突谈判
```
原则：
- 分离人与问题
- 基于利益而非立场
- 创造双赢

话术：
- "我们能否换个角度..."
- "我想知道您的需求是..."
- "有什么方案能满足..."
```

## 谈判策略选择

### 竞争策略
```
适用：
- 一次性关系
- 强势地位
- 充分准备

技巧：
- 开场强硬
- 坚持立场
- 最小让步
```

### 合作策略
```
适用：
- 长期关系
- 互利共赢
- 双方强势

技巧：
- 开放态度
- 信息共享
- 创造价值
```

### 妥协策略
```
适用：
- 时间压力
- 平衡关系
- 各退一步

技巧：
- 寻找中间点
- 交换让步
- 快速达成
```

### 规避策略
```
适用：
- 时机不对
- 准备不足
- 风险过高

技巧：
- 延期讨论
- 转移焦点
- 寻求帮助
```
