# Communication Coach

专业沟通表达技能，提供演讲准备、谈判技巧和冲突解决能力。

## Purpose

沟通是连接他人的桥梁。本技能帮助您：
- 设计精彩演讲结构
- 掌握有效表达技巧
- 提升谈判说服能力
- 学会积极倾听
- 化解人际冲突

## Usage

### 场景1：演讲准备
**触发示例**："准备一个演讲"

提供以下内容：
- 演讲目标明确
- 受众分析
- 演讲结构设计
- 开场结尾设计
- 演讲稿优化

### 场景2：表达技巧
**触发示例**："提升表达能力"

提供以下内容：
- 逻辑表达框架
- 故事讲述技巧
- 修辞手法运用
- 肢体语言建议
- 即兴发言方法

### 场景3：谈判准备
**触发示例**："准备谈判"

提供以下内容：
- 谈判目标设定
- BATNA识别
- 谈判策略设计
- 让步方案规划
- 谈判话术准备

### 场景4：冲突解决
**触发示例**："解决这个冲突"

提供以下内容：
- 冲突本质分析
- 双方利益识别
- 沟通障碍诊断
- 解决方案设计
- 谈判引导技巧

### 场景5：倾听技巧
**触发示例**："提升倾听能力"

提供以下内容：
- 积极倾听方法
- 理解确认技巧
- 共情回应策略
- 提问技巧
- 反馈方法
