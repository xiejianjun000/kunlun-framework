# 谈判计划案例

## 案例1：薪资谈判计划

### 谈判背景
```
谈判类型：薪资谈判
谈判方：求职者（我） vs HR（公司）
职位：高级软件工程师
期望薪资：25-30万/年
公司报价：20-22万/年
目标：争取26-28万/年
```

### 谈判准备
```
自我分析：
优势：
- 5年相关经验
- 技术栈匹配
- 大厂背景
- 需求紧迫

劣势：
- 公司有备选
- 薪资预算限制
- 无其他offer

BATNA：
- 其他公司offer：24万
- 继续找工作机会
- 延迟6个月决策

市场调研：
- 行业薪资水平
- 同岗位薪资范围
- 公司薪资政策
- 谈判时机
```

### 谈判策略
```
策略框架：

1. 价值呈现
   - 展示技能
   - 项目经验
   - 带来的价值

2. 灵活性
   - 有谈判空间
   - 薪资结构灵活
   - 福利可补充

3. 专业性
   - 数据支持
   - 客观评估
   - 长期视角
```

### 谈判话术准备
```

**开场白**

"感谢您给我这个offer。我对贵公司的业务和团队都非常感兴趣，这是一个很好的机会。"

**表达赞赏**

"我认真审阅了offer，整体方案我都很满意，特别是在职业发展和团队文化方面。"

**提出异议**

"关于薪资部分，我想和您再探讨一下。根据我的调研和市场情况，我的期望在26-28万之间。"

**价值论证**

"我带来了5年的后端开发经验，参与过多个大型项目，熟悉微服务架构。我相信这些经验能够为团队带来直接价值。"

**具体案例**

"在上一家公司，我主导的重构项目，使系统性能提升了40%，这个经验对贵公司的项目会有直接帮助。"

**灵活方案**

"如果26万的预算确实有困难，我们可以考虑其他形式：
- 签字费/项目奖金
- 弹性工作时间
- 股权激励
- 额外福利
- 承诺3个月后根据表现调整"

**倾听策略**

- 先让对方说明
- 不打断
- 记录关键点
- 识别真实限制

**坚持原则**

"薪资是重要的，但不是唯一的考虑因素。我更看重的是成长空间、团队氛围和项目机会。如果这些方面都很匹配，我相信薪资是可以协调的。"

**创造价值**

"我愿意承担额外责任，比如技术分享、面试参与等，这些都可以为团队创造价值。"

**收尾技巧**

"我真的很想加入团队。希望我们能找到一个双方都满意的方案。"

**备选方案**

- 如果不接受，礼貌询问是否可以重新考虑
- 要求书面确认
- 保留后续联系的可能
```

### 谈判模拟
```

模拟场景1：公司坚持22万

我："我理解公司有薪酬体系。22万确实低于我的预期，但我真的很喜欢这个机会。我们可以考虑这样：
- 第一年24万
- 第二年根据表现调整到26-28万
- 如果超出预期，会有项目奖金"

模拟场景2：公司说没有预算

我："我理解预算限制。那我们可以这样：
- 保持22万基础
- 增加10%弹性工作时间奖金
- 提供培训预算
- 弹性工作安排
- 6个月后重新评估"

模拟场景3：公司说其他候选人接受22万

我："谢谢您的坦诚。我需要考虑一下。但我真的很喜欢这个机会。如果可能的话，我愿意在22万的基础上，考虑以下补充：
- 更多的技术挑战机会
- 更大的成长空间
- 更早的表现评估调整
```

### 谈判脚本
```

**开场**
"感谢offer！我对公司很感兴趣，有几个方面想和您确认一下。"

**薪资讨论**
"关于薪资，offer上是22万。根据我的经验和市场调研，我的期望是26-28万。"

**等待回应**

**如果对方说：预算有限**

"我理解。那我们可以考虑其他形式吗？比如：
- 项目奖金
- 额外福利
- 弹性工作
- 提前评估"

**如果对方说：需要请示**

"没问题，我理解流程。需要多长时间？我在这期间可以继续考虑。"

**如果对方说：可以商量**

"太好了！那我们看看具体方案。我的底线是26万，但为了表达诚意，我可以考虑其他福利。"

**收尾**
"无论结果如何，我都希望保持良好的关系。这个机会我非常看重。"
```

### 谈判记录
```

记录要点：
- 公司名称
- 联系人
- 职位
- 报价
- 讨论内容
- 对方立场
- 限制因素
- 下次沟通时间

跟踪：
- 记录要点
- 及时更新
- 下一步计划
- 截止日期
```
