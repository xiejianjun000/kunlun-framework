# 🦞 飞书授权配置指南 - 一键复制版

## 任务
在飞书中创建一个成长知识库，并配置自动授权功能。

---

## 第一步：创建飞书应用

1. 打开 [飞书开放平台](https://open.feishu.cn/)
2. 点击「创建应用」→ 填写应用名称（如：XXX的AI助手）
3. 获取 `App ID` 和 `App Secret`

---

## 第二步：开通权限

在应用管理页面 → **权限管理** → 搜索并开通以下权限：

### 基础权限（必开）
| 权限名称 | 说明 |
|---------|------|
| `contact:user.base:readonly` | 读取用户基本信息 |
| `im:message:send_as_bot` | 以机器人身份发消息 |
| `im:message:send_as_user` | 以用户身份发消息 |

### 知识库权限（创建知识库需要）
| 权限名称 | 说明 |
|---------|------|
| `wiki:wiki` | 知识库读写 |
| `wiki:wiki:write` | 创建知识库页面 |
| `wiki:node` | 知识库节点读取 |
| `wiki:node:write` | 创建知识库节点 |

### 其他可选权限（根据需要）
| 权限名称 | 说明 |
|---------|------|
| `calendar:calendar:readonly` | 读取日历 |
| `calendar:event:create` | 创建日程 |
| `task:task` | 任务权限 |
| `bitable:bitable` | 多维表格权限 |

---

## 第三步：发布版本

权限开通后，必须：
1. 点击「创建新版本」
2. 填写版本号（如 1.0.0）和更新说明
3. 点击「发布」

---

## 第四步：配置 OpenClaw

在 `openclaw.json` 中添加以下配置：

```json
{
  "channels": {
    "feishu": {
      "appId": "cli_这里填你的AppID",
      "appSecret": "这里填你的AppSecret",
      "streaming": true,
      "dmPolicy": "allowlist",
      "groupPolicy": "allowlist",
      "allowFrom": ["ou_这里填你的用户ID"]
    }
  },
  "tools": {
    "alsoAllow": [
      "feishu_wiki_space",
      "feishu_wiki_space_node",
      "feishu_create_doc",
      "feishu_update_doc",
      "feishu_fetch_doc",
      "feishu_get_user",
      "feishu_calendar_event",
      "feishu_task_task"
    ]
  }
}
```

---

## 第五步：重启 OpenClaw

```bash
openclaw gateway restart
```

---

## 第六步：开始使用

配置完成后，与 AI 对话时：
1. AI 调用飞书 API 时会自动弹出授权卡片
2. 用户点击授权
3. 授权成功后 AI 继续执行

---

## 快速验证

配置完成后，可以直接对 AI 说：
> "你好！帮我创建一个 XXX 成长知识库"

AI 会自动创建包含以下结构的知识库：
- 📖 目录
- 📝 工作记录
- 🌱 成长日志
- ✅ 任务完成
- 🚀 进化历程
- 💡 经验总结

---

## 常见问题

**Q: 授权卡片不弹出？**
A: 检查应用是否已发布版本，权限是否开通

**Q: 权限提示"未开通"？**
A: 需要在飞书开放平台开通对应权限并发布新版本

**Q: 如何获取用户 ID？**
A: 让用户在你的 AI 中发送消息，从消息 metadata 中获取 open_id
